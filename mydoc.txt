why can't we have the callback function of useeffect async??????
Ans: - Why? Because Reactâ€™s useEffect hook expects a cleanup function returned from it which is called when the component unmounts. 
Using an async function here will cause a bug as the cleanup function will never get called.

useEffect(async () => {
  const users = await fetchUsers();
  setUsers(users);

  return () => {
    // this never gets called, hello memory leaks...
  };
}, []);

so use-
useEffect(() => {
  (async () => {
    const users = await fetchUsers();
    setUsers(users);
  })();

  return () => {
    // this now gets called when the component unmounts
  };
}, []);
